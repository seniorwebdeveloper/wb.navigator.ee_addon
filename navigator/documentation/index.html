Navigator Tag

The "navigator tag" enables you to show your navigation items in a list. Here is the basic syntax:

{exp:navigator}

{nav_title}

{/exp:navigator}

Everything contained between the opening and closing tag will be repeated for each item, so if you want to wrap each item with some markup you'll do this:

{exp:navigator}

<a href="{nav_url}">{nav_title}</a><br />

{/exp:navigator}

Parameters

  • group="1"
  • orderby="nav_title"
  • sort="asc"
  • backspace="6"
  • switch="option_one|option_two"

group=

group="1" group="group_name"

The ID or Group Name of the Navigator Group containing the items you want to show. You can find the ID of each group on the "Group Management" page.

group will default to "1" if nothing is specified.

orderby=

orderby="nav_title"

The "orderby" parameter sets the display order of the items. Setting options for this parameter include:

  • orderby="nav_custom_url"
  • orderby="nav_description"
  • orderby="nav_entry_id"
  • orderby="nav_id"
  • orderby="nav_order"
  • orderby="nav_properties"
  • orderby="nav_template"
  • orderby="nav_title"
  • orderby="nav_title_length"

"nav_title_length" will order the items by the number of characters in the item's title.

orderby will default to "nav_order" if nothing is specified, honoring the custom order specified in the "Item Management" page.

sort=

sort="asc"sort="desc"

The sort order can be ascending or descending. The order will default to "ascending" if nothing is specified.

backspace=

backspace="6"

Backspacing removes characters from the last iteration of the loop. For example, if you put a <br /> tag between each item you'll have this layout:

Home<br />
News<br />
About<br />

You might, however, not want the <br /> tag after the final item. By adding backspacing you can remove it. Simply count the number of characters and spaces in the item you want to remove and add it to the tag. A <br /> tag has 6 characters, so you would do this:

{exp:navigator backspace="6"}

<a href="{nav_url}">{nav_title}</a><br />

{/exp:navigator}

That will produce code like this:

Home<br />
News<br />
About

switch=

switch="option_one|option_two"

This parameter allows you to set two values that will alternate between the result rows. This is commonly used to have alternating colored backgrounds for the result rows. This parameter works in conjunction with the {switch} variable.

{exp:navigator switch="red|blue"}

<a href="{nav_url}" style="background:{switch}">{nav_title}</a><br />

{/exp:navigator}

Variables

Main variables:

  • {nav_title}
  • {nav_url}
  • {nav_properties}
  • {nav_description}

Additional variables:

  • {nav_id}
  • {nav_template}
  • {nav_category}
  • {nav_category_id}
  • {nav_category_word}
  • {nav_url_title}
  • {nav_entry_id}
  • {nav_title_length}
  • {nav_order}
  • {switch}

nav_title

{nav_title}

This variable simply displays the content from the "Item Title" field.

nav_url

{nav_url}

This variable displays the content from the "Custom URL" field, or a URL combined from these 3 fields:

  • Template
  • Category
  • Entry Title

Valid combinations are:

  • Template + Category
  • Template + Entry Title

Any content in the "Custom URL" field will override these combinations.

nav_properties

{nav_properties}

This variable simply displays the content from the "Properties" field.

nav_description

{nav_description}

This variable simply displays the content from the "Description" field.

nav_id

{nav_id}

The ID associated with the item.

nav_template

{nav_template}

This variable displays the selected "Template". (as "template_group/template")

nav_category

{nav_category}

This variable displays the selected "Category".

nav_category_id

{nav_category_id}

This variable displays the ID of the selected "Category".

nav_category_word

{nav_category_word}

This variable displays the "Category URL Indicator" when using "Category Names In Links". (Control Panel Location: Admin > Weblog Administration > Global Weblog Preferences)

nav_url_title

{nav_url_title}

The url_title associated with the selected "Entry Title".

nav_entry_id

{nav_entry_id}

The entry_id associated with the selected "Entry Title".

nav_title_length

{nav_title_length}

The number of characters of the "Item Title". (not the "Entry Title")

nav_order

{nav_order}

The number associated with the default sort order of the items.

switch

{switch}

This variable is dynamically replaced by the appropriate value specified in the switch= parameter. This variable is commonly used to set a CSS class or a background color.

Examples

Here are a few examples of the navigator tag in use:

A simple horizontal list:

{exp:navigator backspace="3"}

<a href="{nav_url}" {nav_properties}>{nav_title}</a> |

{/exp:navigator}

A bit more advanced example:

<ul>

{exp:navigator group="1" switch="odd|even" orderby="nav_title" sort="desc"}

      <li id="item_{nav_id}" class="{switch}">
            <a href="{nav_url}"{nav_properties}>{nav_title}</a>
            <div class="info">{nav_description}</div>
      </li>

{/exp:navigator}

</ul>

The output could look something like this:

<ul>
      <li id="item_1" class="odd">
            <a href="http://www.pmachine.com/" target="_blank">pMachine</a>
            <div class="info">The mothership...</div>
      </li>
      <li id="item_2" class="even">
            <a href="http://www.my-site.com/index.php/site/about/">About</a>
            <div class="info">About this website...</div>
      </li>
</ul>
